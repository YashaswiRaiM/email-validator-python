# Email Validator in Python

This project demonstrates two different approaches to validating email addresses using Python.

## Approaches

### 1. Manual Validation (Logic-based)
- Uses loops, conditions, and string methods
- Checks length, characters, '@', domain, spaces
- Good for understanding fundamentals

### 2. Regex Validation
- Uses regular expressions
- Short, clean, and efficient
- Commonly used in real-world applications

## Tech Used
- Python
- re (Regular Expressions)

## Run
- python manual_validator.py

- python regex_validator.py
